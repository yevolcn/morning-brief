3. Google Stitch 推出 Voice Canvas 语音设计功能

> 🔗 来源：NxCode

4. Google Stitch 引入 DESIGN.md 设计系统格式

Stitch 推出 DESIGN.md 格式，可将设计规则转化为 Agent 友好的 Markdown 格式，支持导入、导出和复用，从任意 URL 自动提取现有设计系统，确保品牌一致性。
